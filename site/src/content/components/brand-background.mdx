## 使い方

ブランドごとに背景表現が異なる（`ai-dash` / `slt-corporate` = Canvas 3D ウェーブ、`sound-laboratory` = CSS 方眼紙 + マウスパララックス）。
`BrandBackground` は `BRAND_BACKGROUNDS` レジストリを参照して自動切り替えするため、複数ブランドを横断する画面では個別の [WaveBackground](/components/wave-background/) / [GridBackground](/components/grid-background/) より本コンポーネントを推奨する。
