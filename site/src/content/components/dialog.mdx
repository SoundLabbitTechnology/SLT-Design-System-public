## 使い方

確認・破壊的操作の再確認・短い通知に使う。長文フォームには使わない。

{/* Do/Don't の出典: docs/L2-components/guides.md#dialog */}
<DoDont
  dos={["破壊的操作は destructive + キャンセルを併置", "confirmLoading で保存中を表示", "Esc で閉じられる（Radix 標準）"]}
  donts={["長文フォームを Dialog に詰め込む", "処理中に閉じられる UI にする", "フォーカストラップなしの独自モーダルを作る"]}
/>
