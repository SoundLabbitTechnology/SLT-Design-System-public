## 使い方

単一行テキスト・メール・パスワード等に使う。複数行入力には [Textarea](/components/textarea/) を使う。複数 field は `.slt-field-stack` で項目間を広く取る。

{/* Do/Don't の出典: docs/L2-components/guides.md#input */}
<DoDont
 dos={["label を必ず付与し control と近接させる", "複数 field は slt-field-stack で項目間を広く", "エラー時は error + errorMessage"]}
 donts={["placeholder のみでラベル代用", "field 同士をラベル間隔と同じ密さで並べる", "色だけ変えてメッセージなし"]}
/>

**状態**: `error` / `disabled` / `readOnly`。`layout="horizontal"` でラベル横並び。focus はキーボード操作で確認できる。
