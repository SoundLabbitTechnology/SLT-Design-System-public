## 使い方

複数の選択肢からひとつの項目だけを選ぶときに使う。任意項目では「あてはまるものはない」など無選択を表す選択肢を用意する。

<DoDont
  dos={["同じ name の Radio を RadioGroup（fieldset）でまとめる", "任意選択では無選択を意味する選択肢を設ける"]}
  donts={["複数回答できる設問に使う（Checkbox を使う）", "選択解除できない仕様のまま任意項目にする"]}
/>

**状態**: `disabled` / `error` / `checked`。キーボードで矢印キーによるグループ内移動と Space 選択を確認できる。
