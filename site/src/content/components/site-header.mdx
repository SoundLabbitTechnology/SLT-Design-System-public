## 使い方

プロダクト共通のトップナビゲーションシェルとして使う。`logo` / `nav` / `actions` スロットで構成する。

{/* Do/Don't の出典: docs/L2-components/guides.md#siteheader */}
<DoDont
  dos={["logo / nav / actions スロットで composition する", "SiteHeaderLink で現在地表示（active）する", "本文揃えは contentMaxWidth と --site-header-padding-x で行う"]}
  donts={["DS 内にプロダクト固有 URL をハードコードする", "モバイル用に別リンクセットを二重定義する", "上下 padding でヘッダーを膨らませる・Docs でコンポーネント CSS を上書きする"]}
/>
