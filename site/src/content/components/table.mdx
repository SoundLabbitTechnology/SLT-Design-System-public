## 使い方

Admin / ダッシュボードの一覧データに使う。複雑なフィルタ UI は Table に内包しない。

{/* Do/Don't の出典: docs/L2-components/guides.md#table */}
<DoDont
 dos={["DataTable でソート・選択の最小 API を使う", "TableHead の sortable + aria-sort を使う", "TableCheckboxCell で行選択する"]}
 donts={["複雑なフィルタ UI を Table に内包する", "ソート可能な見た目だけにする（ボタンなし）", "行クリックのみで選択する（キーボード不可）"]}
/>

**状態**: ソート asc/desc、行選択、ヘッダー全選択（indeterminate 対応）、`striped`、セル結合（`colSpan` / `rowSpan`）。外周枠線あり。
