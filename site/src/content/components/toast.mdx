## デモ

ボタンを押すとトーストが右下に表示されます。

<Demo>
 <ToastDemo client:visible slot="stage" />
 <Fragment slot="code">
 ```tsx
 import { ToastProvider, useToast } from "@soundlabbit/design-system/ui";

 <ToastProvider>
 <App />
 </ToastProvider>

 const { toast } = useToast;
 toast({ title: "保存しました", variant: "success" });
 ```
 </Fragment>
</Demo>

## 使い方

操作結果の短いフィードバック（保存完了・エラー通知）に使う。重要な確認には [Dialog](/components/dialog/) を、常時表示のステータスには [Badge](/components/badge/) やインライン文言を使う。

{/* Do/Don't の出典: docs/L2-components/guides.md#toast */}
<DoDont
 dos={["アプリルートを ToastProvider でラップする", "成功 / 警告 / エラーに variant を使う", "複数件はスタックで表示する（自動）"]}
 donts={["Provider なしで useToast を呼ぶ", "長文やフォーム全体を Toast に入れる", "同時に大量の Toast を発火する"]}
